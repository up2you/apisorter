# Windows 10 簡易安裝指南 (使用內建資料庫)

這份指南將協助您在全新的 Windows 10 電腦上，使用最簡單的方式安裝並執行 API Sorter。我們將使用內建的資料庫 (Embedded Postgres)，因此**不需要**安裝 Docker。

## 1. 系統前置準備

在開始之前，您只需要安裝 **Node.js**。

### 安裝 Node.js
1.  前往 [Node.js 官方網站](https://nodejs.org/)。
2.  下載 **LTS (長期支援)** 版本 (例如 v20.x 或 v22.x)。
3.  執行安裝程式，一路點擊 "Next" 直到完成。
4.  安裝完成後，開啟「命令提示字元 (CMD)」或 PowerShell，輸入以下指令確認安裝成功：
    ```bash
    node -v
    npm -v
    ```
    如果有出現版本號碼 (例如 `v20.11.0`)，代表安裝成功。

---

## 2. 下載與安裝專案

### 方法 A: 使用一鍵安裝腳本 (推薦)

如果您已經取得了專案檔案 (例如解壓縮了 zip 檔，或使用 git clone 下載)：

1.  進入專案資料夾 `apisorter_ag`。
2.  找到檔案 **`setup_and_run.bat`**。
3.  **雙擊執行**該檔案。

這個腳本會自動執行以下動作：
*   檢查 Node.js 是否安裝。
*   安裝所有必要的程式套件 (`npm install`)。
*   建立設定檔 (`.env`)。
*   在一個新視窗中啟動內建資料庫。
*   啟動網站伺服器。

當您看到 **`Ready in ...`** 字樣時，打開瀏覽器訪問 [http://localhost:3000](http://localhost:3000) 即可使用！

---

### 方法 B: 手動安裝 (進階)

如果您希望手動執行每個步驟，請依序執行以下指令：

1.  **安裝套件**
    開啟終端機 (Terminal) 並在專案目錄下執行：
    ```bash
    npm install
    ```

2.  **設定環境變數**
    複製範例設定檔：
    ```bash
    copy .env.example .env
    ```
    (可選) 使用記事本編輯 `.env` 檔案，填入您的 Google Gemini API Key (若需要 AI 功能)。

3.  **啟動資料庫**
    在一個新的終端機視窗中執行：
    ```bash
    node scripts/dev-db.mjs
    ```
    請保持這個視窗開啟，不要關閉。

4.  **啟動網站**
    在原本的終端機視窗中執行：
    ```bash
    npm run dev
    ```

---

## 3. 關於 AI 功能

本系統的基礎功能 (瀏覽、搜尋、排序 API) **完全免費且不需要外部服務**。

唯有以下進階 AI 功能需要設定 API Key 才能使用：
*   自動翻譯 API 描述
*   智慧搜尋建議
*   自動分類

若您需要這些功能，請編輯專案根目錄下的 `.env` 檔案，並填入您的 `GOOGLE_API_KEY`。
